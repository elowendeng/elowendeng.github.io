## 📋 About

This is my personal academic website with a clean two-column layout, featuring personal information, research interests, news updates, academic experience, and more. The site supports **dark/light mode toggle** and **real-time search functionality**.

### ✨ Features

- **Clean and Modern Design** - Two-column layout: fixed personal information on the left, scrollable content on the right.

- **Dark/Light Mode** - One-click switching, automatically remembers user preferences.

- **Real-time Search** - Search for personal experiences, news, contact information, etc.

- **Responsive Design** - Perfectly adapts to desktop and mobile devices.

- **GitHub Pages Deployment** - Fast, free, and stable.

## 🚀 Run Locally

1. **Clone the repository**
   
   ```bash
   git clone https://github.com/elowendeng/elowendeng.github.io.git
   ```
   
2. **Navigate to project directory**
   
   ```bash
   cd elowendeng.github.io
   ```
   
3. **Add your photo**
   
   - Name your photo `photo.jpg` and place it in the `images/` folder.
   
4. **View locally**
   
   - Open the `index.html` file directly in your browser.

   - Or use Live Server (VS Code plugin) to start a local server.

## 🎨 Customization Guide

### Edit Personal Information

In `index.html`, find the following sections and modify them:

- Name, School, Lab

- Contact Information (Email, GitHub, LinkedIn)

- Self-introduction

- Personal Experience

### Edit Search Data

Add or modify search entries in the `searchData` array within the JavaScript section:

```javascript
const searchData = [
    { title: "Your Title", category: "Category", content: "Content Description" },
    // Add more...
];
```

### Modify the Color Theme

Adjust CSS variables in the `<style>` tag:

- Light mode primary color: `#3b82f6`

- Dark mode primary color: `#60a5fa`
